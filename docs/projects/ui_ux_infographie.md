# Créer une infographie UI vs UX (180 points)

## Objectif
Créer une infographie claire et visuelle qui explique la différence entre UI (User Interface) et UX (User Experience) Design.

## Recherche et compréhension

Avant de commencer, assurez-vous de bien comprendre les concepts :

**UI (User Interface Design)**
- Se concentre sur l'apparence visuelle
- Éléments : couleurs, typographie, boutons, icônes
- Objectif : rendre l'interface esthétique et cohérente

**UX (User Experience Design)**
- Se concentre sur l'expérience globale de l'utilisateur
- Éléments : recherche utilisateur, architecture de l'information, parcours utilisateur
- Objectif : rendre le produit intuitif et facile à utiliser

**Questions à explorer :**
- Quelles sont les 3 principales différences ?
- Comment ces deux disciplines collaborent-elles ?
- Quels outils utilisent respectivement les designers UI et UX ?

## Planification du contenu

Décidez quelles informations inclure dans votre infographie :

**Sections suggérées :**
1. Titre accrocheur
2. Définitions courtes de UI et UX
3. Comparaison visuelle (tableau ou diagramme)
4. Métaphore illustrative (ex : UI = décoration d'une maison, UX = plan architectural)
5. Outils populaires pour chaque discipline
6. Citation ou fait intéressant

**Conseil :** Limitez le texte. Une infographie efficace est 70% visuelle, 30% texte.

## Création de la structure

Choisissez votre outil :
- Canva (gratuit, facile)
- Figma (professionnel)
- Adobe Illustrator (avancé)
- PowerPoint ou Google Slides (accessible)

**Structure recommandée :**
- Format vertical (800 x 2000 pixels minimum)
- Divisez l'espace en 5-6 sections horizontales
- Utilisez une grille pour aligner les éléments

**Palette de couleurs :**
- Choisissez 2-3 couleurs principales
- Suggestion : une couleur pour UI, une pour UX, une neutre pour le texte

## Design visuel

### En-tête
- Titre principal : "UI vs UX : Quelle est la différence ?"
- Sous-titre optionnel
- Icône ou illustration d'introduction

### Corps principal
Créez une section comparative avec :

**Colonne UI :**
- Icône représentative (palette, écran, pinceau)
- Définition en 1-2 phrases
- 3-4 mots-clés (ex : visuel, interface, esthétique, branding)
- Exemple d'outils (Sketch, Figma, Adobe XD)

**Colonne UX :**
- Icône représentative (personne, parcours, carte)
- Définition en 1-2 phrases
- 3-4 mots-clés (ex : recherche, ergonomie, parcours, wireframes)
- Exemple d'outils (Miro, Maze, UserTesting)

### Section métaphore
Illustrez avec une analogie visuelle :
- **Restaurant :** UI = la décoration et le menu, UX = la qualité du service et l'expérience
- **Voiture :** UI = le design extérieur, UX = le confort de conduite
- Créez votre propre métaphore originale !

### Collaboration
Montrez comment UI et UX travaillent ensemble :
- Diagramme de Venn
- Ou flèche cyclique montrant le processus itératif
